## Descrição da Ação
São atualizados alguns campos de acordo com a origem do financeiro:

Se for contrato, a data de vencimento é atualizada de acordo com a regra correta.
Se for das notas da grafeno, são atualizados os campos de conta, banco, data de vencimento e tipo de título.
